# Business Landing Page

Професійна landing page для бізнесу, розроблена на базі дизайну з Figma. Проект демонструє сучасний підхід до створення привабливого вебсайту для залучення клієнтів.

## Про проект

Це landing page, розроблена на **HTML, CSS та JavaScript**. Сайт розроблений за дизайном, представленим у [Figma прототипі](https://www.figma.com/design/ccQ412S8bV6FenLPWOZYGW/Figmaland--Business-Landing-page--Community---Community-?node-id=65-122&t=Z6o5uQoErfwGIlXt-0).

## Функціонал

- Адаптивний дизайн (responsive design)
- Сучасний та привабливий інтерфейс
- Швидкої завантаження
- Чистий і легкоуразумінний код

## Технологічний стек

- **HTML5** - семантична розмітка
- **CSS3** - стилізація та адаптивність
- **JavaScript (ES6+)** - інтерактивність

## Установка та запуск

### Локальне розгортання

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/anna-fedorenko/business-landing-page.git
   cd business-landing-page
   ```

2. **Відкрийте файл у браузері:**
   
   Просто відкрийте файл `index.html` у вашому браузері або використовуйте локальний сервер:
   ```bash
   # Якщо встановлено Python 3
   python -m http.server 8000
   
   # або з Node.js (http-server)
   npx http-server
   ```

   Потім перейдіть на `http://localhost:8000`
